Detailed Description

The Pokemon class represents a Pokémon in the game.

It contains attributes such as name, type, stats, moves, and status conditions. The class provides methods to manage the Pokémon's stats, abilities and interactions with other Pokémon. All interactions are followed by an Event Observer pattern to handle game logic during battles.

Returns
boolean False if status prevents action, true otherwise
