Detailed Description

This enum represents the different status effects that can be applied to Pokémon during battles.

Each status effect is associated with a specific method suffix that indicates the type of effect it has. "Condition" refers to status conditions that a pokémon can be in, while "Attack", "Defense", "SpAttack", "SpDefense", "Speed", "Accuracy" refer to the stats that can be affected.
