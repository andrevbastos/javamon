Detailed Description

Manages multiple simulation states using Composite pattern.

This class implements SimulationState and serves as a container for managing all simulation states collectively.

  • Maintains a collection of SimulationState objects
  • Provides centralized reset functionality
  • Allows easy addition of new states
Note
While implementing SimulationState, most methods are empty as this class primarily manages child states
