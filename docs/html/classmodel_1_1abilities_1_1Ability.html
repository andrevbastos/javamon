Detailed Description

Abstract base for all Pokémon abilities.

Implements Template Method pattern for battle effects. Each concrete ability defines execution logic for its event.

Key features:

  • Owner Pokémon reference
  • Associated trigger event
  • Modifiable battle parameters
  • Status effect handling
Note
Subclasses must implement execute() method
