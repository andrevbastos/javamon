Detailed Description

The Combat class is responsible for simulating battles between Pokémon.

It manages the selected Pokemon and instantiates them using the PokemonFactory class. The class handles the battle logic, including determining the order of moves, logging the alterations and determining the winner. It uses the AbilityObserver to handle events during the battle.
