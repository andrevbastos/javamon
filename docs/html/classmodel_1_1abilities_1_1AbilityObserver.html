Detailed Description

Mediates ability-triggered interactions between battling Pokémon.

Acts as the central hub for ability activation during battles. Implements the Observer pattern to handle ability events between two Pokémon.

  • Tracks two battling Pokémon (p1 and p2)
  • Dispatches events to appropriate abilities
  • Handles four event types with method overloading
Note
Thread-safe through AtomicReference for damage multipliers
